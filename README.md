# UTM Desktop Releases

Download the latest version of UTM Desktop from the [Releases](https://github.com/josephdaw/utm-desktop-releases/releases) page.

## Installation

### macOS

1. Download the `.dmg` file from the latest release
2. Open the `.dmg` and drag UTM Desktop to your Applications folder
3. **Important:** The app is not yet signed with an Apple Developer certificate. On first launch:
   - macOS will show a warning that it "cannot verify the developer"
   - Go to **System Settings â†’ Privacy & Security**
   - Scroll down and find "UTM Desktop was blocked"
   - Click **Open Anyway**
   - The app will now run normally

### Windows

1. Download the `.exe` file from the latest release
2. Run the installer and follow the prompts
3. Windows may show a SmartScreen warning - click "More info" then "Run anyway"

## Update Channels

- **Stable releases** - Recommended for most users
- **Pre-releases (Beta)** - Early access to new features, may contain bugs

You can switch between channels in the app's Update Settings.

## Auto-Updates

Once installed, the app will automatically check for updates. You can also manually check via the Updates section in the app.
